A collection of simple, beautiful demo sites for Woostify.

== Description ==
This plugin is an add-on for the Woostify WordPress Theme. It gives users a collection of demo sites to choose and import. Here is how it works:

1. Install required plugins.
2. Choose a demo site to import.
3. Import site content and media.

More info: https://woostify.com

Done!

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "Woostify Sites Library" into the search box and hit Enter.
3. Locate the Atomic Blocks plugin in the list of search results and click **Install Now**.
4. Once installed, click *Activate*.
5. Navigate to _Appearance &rarr; Woostify Sites Library_ to import demo sites.

It can also be installed manually.

1. Download the Woostify Sites Library from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Woostify Sites Library in the list and click *Activate*.
5. Navigate to _Appearance &rarr; Woostify Sites Library_ to import demo sites.
